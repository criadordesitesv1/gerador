📋 Como gerar seu Token do GitHub

1
Acesse github.com/settings/tokens/new
2
Em Note, escreva: gerador-sites
3
Em Expiration, escolha No expiration
4
Marque a caixa repo (primeira da lista)
5
Clique em Generate token e copie o código
```
⚡ Instalar Gerador
```
